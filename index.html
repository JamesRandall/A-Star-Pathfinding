A* Algorithm

This is a HTML 5 Canvas and JavaScript implementation of the A* path finding algorithm - in this example only horizontal or vertical movement is allowed. It makes use of the JavaScript libraries jQuery and RequireJS.

Source code can be found online at GitHub and more information on my blog.

You can a predetermined board layout below or edit the board by clicking cells.

Your browser does not support running A*. Please upgrade to a more recent version that supports HTML 5 canvas.
